Installation
============

Complex Sorter:
Copy "FO4Edit\Edit Scripts"-files to the "FO4Edit\Edit Scripts" folder. 
Start the M8r_ComplexSorter.pas via 'Apply Script' link in context menu.

Tag configuration: 
You will need a tag configuration thats compatible with your Item Sorting. If you are going to create
a normal FallUI Item Sorter sorting, then you can simply use the included package in the FallUI Inventory FOMOD installer. 
Alternatively use the tag configuration in the folder "FallUI - Item Sorter Mod tag configuration". For installation, 
just copy the contents of that folder to your Fallout 4 data folder (or create a mod of that folder).

See mod page on Nexus for more instructions.
